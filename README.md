# cookieJar()-backend : A Description
#### Our team set out to create an e-commerce application that allows users to purchase cookies. We're still working on integrating Stripe.

## Useful Links:

## User Stories:
### Version 1 ERD
#### User
- email
- hashedPassword
- token

#### Purchases
- item_name : string
- item_price : number
- date_of_purchase : date
- review: string
- owner: ref to user
